# SMOTE Demo

This Streamlit app demonstrates how SMOTE creates synthetic examples to balance an imbalanced classification dataset.

The app allows students to:

- choose between a synthetic imbalanced dataset and the breast cancer dataset;
- inspect class distribution before SMOTE;
- visualise the data before SMOTE;
- apply SMOTE;
- inspect class distribution after SMOTE;
- visualise the data after SMOTE;
- download the resampled dataset.

## Scientific reference

Chawla, Nitesh V., Kevin W. Bowyer, Lawrence O. Hall, and W. Philip Kegelmeyer (2002). “SMOTE: Synthetic Minority Over-Sampling Technique.” *Journal of Artificial Intelligence Research*, 16, 321–357.

Lemaître, Guillaume, Fernando Nogueira, and Christos K. Aridas (2017). “Imbalanced-learn: A Python Toolbox to Tackle the Curse of Imbalanced Datasets in Machine Learning.” *Journal of Machine Learning Research*, 18(17), 1–5.
