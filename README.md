# SMOTE Demo

This Streamlit app demonstrates how SMOTE creates synthetic examples to balance an imbalanced classification dataset.

The app allows to:

- choose between a synthetic imbalanced dataset and the breast cancer dataset;
- inspect class distribution before SMOTE;
- visualise the data before SMOTE;
- apply SMOTE;
- inspect class distribution after SMOTE;
- visualise the data after SMOTE;
- download the resampled dataset.

## Scientific reference

N. V. Chawla, K. W. Bowyer, L. O. Hall, and W. P. Kegelmeyer (2002). “SMOTE: Synthetic Minority Over-Sampling Technique.” *Journal of Artificial Intelligence Research*, 16.
